== Description ==

WooCommerce PDF Invoice Bulk Download plugin allows you to download WooCommerce PDF invoices in bulk as zip files. You can filter and group invoices by date range, months order status and much more. The plugin currently supports the following WooCommerce PDF Invoice plugins:
[WordPress](http://wordpress.org/ "Your favorite software")
*   WooCommerce PDF Invoices - By Bas Elbers
*   WooCommerce PDF Invoices & Packing Slips
*   WooCommerce PDF Invoices - By Andrew Benbow
*   RightPress WooCommerce PDF Invoice - By RightPress

If you're facing any issues downloading your PDF invoices as ZIP archives with plugins named above please contact us at [support@havealooklabs.com]("mailto:support@havealooklabs.com")

Features:

*   Download WooCommerce PDF invoices in bulk as ZIP archives
*   Download WooCommerce PDF invoices per month, date range and year
